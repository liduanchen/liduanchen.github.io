# 如何开始学习编程

编程看起来很难，但其实入门并不复杂。让我分享一些学习经验，希望能帮助想要学习编程的朋友。

### 1. 选择一门语言

对于初学者，我推荐 **Python** 或 **JavaScript**。
- **Python**: 语法接近自然语言，适合数据分析、AI 和自动化。
- **JavaScript**: 网页开发的基石，可以看到即时的反馈。

### 2. 动手写代码

不要只看视频！看一百遍教程不如自己写一行代码。
建议去 LeetCode 或 Codecademy 这种互动平台。

### 3. 寻找一个小项目

比如做一个待办事项清单，或者...搭建一个自己的博客！
项目驱动学习是最快的方式。
