# ✍️ 博客文章更新指南

欢迎使用你的全新现代化博客系统！现在，你不再需要手动编写 HTML 页面，只需编写 Markdown 文件并更新一个简单的 JSON 文件即可。

---

## 🚀 步骤 1：编写文章 (Markdown)

1. 进入 `posts/` 目录。
2. 创建一个新的 `.md` 文件（例如：`my-new-post.md`）。
3. 使用 Markdown 语法编写你的内容。你可以包含代码块、列表、图片等。

   **示例内容：**
   ```markdown
   # 这是我的新文章标题

   这是文章的第一段内容。

   ### 小标题
   - 列表项 1
   - 列表项 2
   ```

---

## 📂 步骤 2：注册文章 (JSON)

为了让首页能够看到你的新文章，你需要将其信息添加到根目录下的 `posts.json` 文件中。

1. 打开 `posts.json`。
2. 在数组中添加一个新的对象：

   ```json
   {
     "id": "my-new-post",          // 唯一标识符（建议与文件名一致，不要有空格）
     "title": "这是我的新文章标题",     // 首页显示的标题
     "date": "2026-01-20",         // 发布日期
     "excerpt": "这是文章的简短摘要...", // 首页显示的预览文字
     "category": "随笔",            // 分类（例如：随笔、教程、生活）
     "filename": "my-new-post.md"  // 你在步骤 1 中创建的文件名
   }
   ```
   *注意：请确保新添加的对象与前面的对象之间有逗号分隔。*

---

## 🎨 步骤 3：预览与发布

1. **本地预览**：在浏览器中打开 `index.html`（由于使用了 `fetch` 加载数据，建议通过本地服务器预览，你可以在 VS Code 中使用 `Live Server` 插件）。
2. **发布**：将你的改动（新的 `.md` 文件和更新后的 `posts.json`）上传到你的 GitHub 仓库。GitHub Pages 会自动更新你的博客。

---

## 💡 进阶技巧

- **分类过滤**：如果你在 `posts.json` 中定义了新的 `category`，你可以根据需要在 `index.html` 的 `.category-filter` 区域添加对应的按钮。
- **图片**：将图片存放在 `images/` 目录中，然后在 Markdown 中通过 `![描述](images/your-image.png)` 引用。
- **暗色模式**：点击导航栏右侧的 🌙 按钮即可切换主题，系统会自动保存你的选择。
